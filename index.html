// About
How It Works
01
Reads Prediction Markets Jupiter API
Prediction markets are pools where people stake money on future outcomes — "Will SOL hit $200 this month?" The agent polls these every 15 minutes and tracks probability shifts, not absolute odds. A jump from 40% to 65% YES signals far more than odds sitting at 65% for weeks. Crowd conviction changing is the signal.
02
Checks Token Health Jupiter Tokens API
Cross-references crowd sentiment against on-chain reality: organic trading volume (not bot-driven), real holder count, and whale concentration risk. This prevents acting on manipulated markets. Whale flag detected? Buy multiplier is hard-capped at 0.8× regardless of signal score.
03
AI Combines Signals Claude claude-sonnet-4-6
Claude scores both inputs into a 0–100 composite signal with contextual weighting. A 10pp market shift means more when volume is also spiking — Claude handles that nuance. Score above 70 = BULLISH. Below 30 = BEARISH, buys paused. The AI also determines optimal TP/SL percentages for each bracket.
04
Adjusts Your DCA Jupiter Recurring API
DCA buys a fixed amount on a schedule. Market Whisperer makes it adaptive — bullish signal buys 1.4–2× more per cycle, bearish signal pauses entirely. You buy more when the crowd is confident, less when they're scared. The schedule is rebuilt each cycle based on the current signal.
05
Auto Brackets Every Buy Jupiter Trigger API
Every DCA entry automatically gets an OCO (One Cancels Other) bracket: a Take Profit and a Stop Loss. When one triggers, the other cancels automatically. No monitoring needed. The ratio is always asymmetric (minimum 1.3:1 TP:SL) so winners run further than losers cut.
06
Executes on Solana Jupiter Swap V2
All swaps route through Jupiter Swap V2 — best price aggregator on Solana. Finds the cheapest path across all DEXes automatically. Gasless swaps built in. The /order + /execute flow handles optimal landing and MEV protection.
